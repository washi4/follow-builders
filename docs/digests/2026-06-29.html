Anthropic Engineering

如何在不同产品中限制 Claude (How we contain Claude across products)
Anthropic 详细分享了如何在 claude.ai、Claude Code 和 Claude Cowork 中设计安全隔离架构，以限制 AI agent 的爆炸半径。
• 环境层隔离（如沙盒、虚拟机）比依赖模型层监督或人类审批更可靠，因为用户容易产生审批疲劳，而模型也可能被提示注入绕过。
• 文章披露了几个真实的安全教训：例如 Claude Code 在用户确认信任前就解析了恶意配置，以及 Claude Cowork 中攻击者利用允许的 Anthropic API 域名成功渗出数据。
• 核心原则是：首先在设计上限制环境权限，其次才是引导模型行为；同时，隔离强度必须与用户的技术监督能力相匹配。
