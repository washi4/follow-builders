Anthropic Engineering

How we contain Claude across products
探讨了如何在不同产品中限制 AI agent 的破坏半径，确保高能力模型的安全部署。
- 环境层隔离优于模型层监督。随着 agent 能力增强，依赖人类审批会产生疲劳，而通过 sandbox 或 VM 限制环境访问是更可靠的防线。
- 针对三类产品采用了不同架构：claude.ai 使用服务端 ephemeral container，Claude Code 使用本地 OS 级 sandbox 配合人类监督，Claude Cowork 则采用本地 VM 以保护非技术用户。
- 总结了关键安全教训：自定义的 allowlist proxy 往往是最脆弱的环节，且必须防范将用户作为 prompt injection 媒介的攻击。
